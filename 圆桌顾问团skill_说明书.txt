# 四大投资大师圆桌顾问团 · 使用说明书

---

## 一、这是什么

将「圆桌顾问团skill.txt」的全部内容粘贴到任意支持自定义系统提示（System Prompt）的 AI 工具中，即可激活四大投资大师圆桌顾问团。

激活后，**每次提问都是一场圆桌会议**：
- 四位大师依次详细作答，主动指出彼此的分歧
- 无论是否附带年报数据，格式始终一致
- 最终输出完整的投资结论与分阶段操作建议

| 大师 | 核心身份 |
|------|---------|
| 🔵 格雷厄姆 | 价格与安全边际的守护者，数字驱动，最保守 |
| 🟡 费雪 | 管理层与成长质量的判断者，定性调研，最重视人 |
| 🟠 芒格 | 失败路径与激励机制的洞察者，反向思考，最犀利 |
| 🔴 巴菲特 | 护城河与资本配置的评估者，综合框架，最系统 |

---

## 二、如何提问

**直接提出投资或公司相关问题即可，无任何格式要求。**

可以附带年报数据，也可以不附带——有数据时分析更深入具体，没有数据时四位大师基于原则框架给出定性判断。

---

## 三、附带年报数据时的格式

建议提供 **3年起步，5年最佳**。覆盖一个完整经济周期，才能进行最差年份压力测试和趋势分析。

**三张报表数据越完整，分析越精确：**

```
XX公司 2020-2024 年报数据：

【利润表】
营收：50 / 65 / 80 / 88 / 92 亿元
毛利润：15 / 20 / 26 / 27 / 28 亿元
净利润：5 / 7 / 9 / 8 / 10 亿元

【现金流量表】
经营现金流：6 / 8 / 10 / 7 / 11 亿元
资本支出：3 / 4 / 5 / 6 / 5 亿元

【资产负债表】
总资产：80 / 95 / 120 / 140 / 160 亿元
有息负债：10 / 12 / 15 / 18 / 16 亿元
股东权益：40 / 45 / 55 / 60 / 72 亿元
大股东持股：45% / 44% / 44% / 43% / 43%

当前股价：25元，PE：15倍，PB：2倍
```

也可以直接粘贴年报原文段落，或用自然语言描述，AI会自动提取关键信息。

## 四、每次提问会得到什么

### 四位大师的独立分析（必须全部作答）

每位大师的分析包含：
- 从各自框架出发的详细判断
- **⚔️ 主动指出与前位大师的分歧及根本原因**
- 一句符合其人格的风格语录
- 明确的操作结论（买入/观望/回避等）

**有年报数据时，每位大师额外输出：**

| 大师 | 三张报表的分析重点 |
|------|-----------------|
| 🔵 格雷厄姆 | 利润表最差年份压力测试 · 现金流/净利润含金量逐年 · 资产负债表NCAV计算 · 七项防御性检验逐条 · 给出买入价上限 |
| 🟡 费雪 | 利润表毛利率趋势与成长驱动力 · 现金流自由现金流与融资依赖判断 · 资产负债表大股东持股逐年变化 · 管理层预测兑现率 · 十五点评分 |
| 🟠 芒格 | 利润表定价权量化验证 · 现金流激励扭曲痕迹识别 · 资产负债表失败路径识别 · 年报措辞心理学审查 |
| 🔴 巴菲特 | 利润表所有者盈余逐年计算 · 现金流资本配置历史评分 · 资产负债表护城河资产表现 · 七大红旗逐项清查 · 内在价值估算给出买入区间 |

---

### 圆桌投资结论（每次必须输出）

包含四个部分：

**一、四位大师综合建议汇总表**
操作建议 + 信心等级（⭐～⭐⭐⭐⭐⭐）+ 核心理由 + 最大风险

**二、分歧汇总与根本原因**
所有大师之间的分歧点整合，每个分歧附解释根本原因

**三、分阶段投资操作建议**
- 阶段一：初步建仓条件与仓位
- 阶段二：加仓条件与累计仓位
- 阶段三：满仓条件（若大师高度认可）
- 卖出/止损的具体触发条件

**四、年报关键要点总结**（仅当提供年报数据时）
✅ 积极信号 / ⚠️ 风险信号 / ❓ 待核实事项

## 五、适用的所有提问类型

### 5.1 个股分析
- "你们怎么看贵州茅台？"（可附带年报数据）
- "腾讯现在值得买入吗？"
- "以下是某公司2020-2024年三张报表数据，请分析是否值得投资"

### 5.2 持仓决策
- "我持有某公司三年，现在竞争对手进来了，继续持有还是卖出？"
- "股价从低点涨了80%，还应该持有吗？"
- "连续三个季度业绩下滑，要不要卖？"

### 5.3 行业研判
- "白酒行业现在值得配置吗？"
- "半导体行业适合价值投资吗？"
- "消费医疗和工业软件，哪个更符合价值投资逻辑？"

### 5.4 投资原则
- "集中持有5只和分散持有20只，哪个更好？"（分歧明显）
- "高股息股票值得长期持有吗？"（格雷厄姆与费雪分歧）
- "用杠杆投资价值股可以吗？"
- "ETF和选股，普通人该选哪个？"

### 5.5 估值与财务
- "PE为30倍算贵吗？"
- "所有者盈余和净利润，哪个更准确？"
- "安全边际要留多少才算够？"
- "经营现金流持续低于净利润，是危险信号吗？"

### 5.6 买卖时机
- "市场大跌30%，现在应该加仓吗？"
- "什么情况下应该卖出一只好股票？"
- "利率上升对价值股意味着什么？"

### 5.7 管理层与公司治理
- "如何判断管理层是否值得信任？"
- "创始人大量减持意味着什么？"
- "公司频繁并购是好事还是坏事？"
- "股权回购是否真的对股东有利？"

### 5.8 投资心理
- "买入后股价跌了40%，该怎么办？"
- "我总是追涨杀跌，如何克服？"
- "怎么建立一套不受市场情绪影响的投资系统？"

## 六、四位大师分歧最明显的话题

| 话题 | 分歧概要 |
|------|---------|
| 成长股溢价 | 格雷厄姆拒绝；费雪接受；芒格/巴菲特视护城河而定 |
| 股息政策 | 格雷厄姆支持分红；费雪反对高成长期高股息 |
| 集中 vs 分散 | 格雷厄姆分散；费雪/芒格/巴菲特均主张集中 |
| 宏观预测 | 格雷厄姆有参考价值；芒格/巴菲特系统性蔑视 |
| 高PE股票 | 格雷厄姆危险信号；费雪认为PE本身不说明贵贱 |
| 科技/成长行业 | 格雷厄姆回避；费雪拥抱；巴菲特/芒格取决于能力圈 |

**分歧处是最有价值的部分**——不是需要"选边站"的地方，而是同一问题从不同维度照进来的光，结合起来比单一视角更可靠。
